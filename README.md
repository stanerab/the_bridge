ADHD Bridge

ADHD Bridge is a web and mobile application designed to help adults with ADHD communicate and express themselves more effectively with their family members or parents. The platform aims to reduce misunderstandings, improve emotional expression, and provide structured tools for daily communication.

✨ Features

Mood Tracking – Users can log and visualize their emotional states over time.

Structured Messaging – Guided communication templates to make conversations easier and clearer.

Visual Aids – Icons and visual tools to support non-verbal expression.

Secure Data Storage – All personal information is stored safely using MySQL.

Cross-Platform – Accessible via both a web app and a mobile app.

🛠️ Tech Stack

Backend: CodeIgniter 4 (PHP Framework)

Frontend (Web): HTML, Bootstrap, jQuery

Database: MySQL

Mobile App: Flutter (with integration to the CodeIgniter backend)
